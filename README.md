# ☕ Java: Aplicando a Orientação a Objetos

> Curso da plataforma [Alura](https://cursos.alura.com.br/course/java-aplicando-orientacao-objetos) — Back-End | Nível Iniciante

---

## 📋 Sobre o Curso

Este curso é voltado para quem já possui noções básicas de Java e deseja dominar o **paradigma de Orientação a Objetos (POO)**, fundamento essencial para o desenvolvimento de aplicações profissionais em Java.

A proposta é evoluir progressivamente um projeto inspirado em uma plataforma de streaming chamada **ScreenMatch**, modelando e refinando suas classes ao longo de cada módulo.

- ⏱️ **Carga horária:** ~8 horas
- 🎯 **Nível:** Iniciante
- 🏫 **Plataforma:** Alura
- 📜 **Certificado:** Sim (reconhecido pelo mercado)

---

## 🎯 Objetivos do Curso

Ao concluir este curso, serei capaz de:

- ✅ Modelar abstrações através de **classes**, **atributos** e **métodos**
- ✅ Aplicar **encapsulamento** com modificadores de acesso e getters/setters
- ✅ Utilizar **herança** para reaproveitar código entre classes relacionadas
- ✅ Aplicar **polimorfismo** para tornar o código mais flexível e extensível
- ✅ Definir contratos com **interfaces** para padronizar comportamentos
- ✅ Construir uma aplicação prática utilizando todos os pilares da POO

---

## 📚 Conteúdo e Módulos

### 1. 🎬 Modelando o ScreenMatch
- Criação do modelo para representar filmes
- Definição de atributos (título, duração, ano de lançamento, etc.)
- Implementação de métodos e comportamentos da classe `Filme`

### 2. 🔒 Controlando o Acesso e a Escrita nos Dados
- Uso de **modificadores de acesso** (`private`, `public`, etc.)
- **Encapsulamento** para proteger os dados dos objetos
- Criação de métodos **getters** e **setters**

### 3. 🔁 Reaproveitando Características e Comportamentos
- Aplicação do conceito de **herança**
- Criação da classe `Serie` reaproveitando código da classe `Filme`
- Introdução ao **polimorfismo** com sobrescrita de métodos

### 4. 🤝 Aplicando Comportamentos em Comum
- Uso de **interfaces** para definir contratos entre classes
- Adição de funcionalidades e comportamentos compartilhados entre diferentes tipos

### 5. 🏋️ Consolidando Conhecimentos
- Desafios práticos de modelagem orientada a objetos
- Criação de aplicações completas aplicando todos os conceitos do curso

## 🚀 Como Executar o Projeto

### Pré-requisitos
- [JDK 17+](https://www.oracle.com/java/technologies/downloads/) instalado
- [IntelliJ IDEA](https://www.jetbrains.com/idea/download/) (ou outra IDE de sua preferência)

### Passos
```powershell
# 1. Clone o repositório
git clone https://github.com/Moquiuti/Java-Aplicando-a-Orienta-o-a-Objetos.git

# 2. Abra o projeto na IntelliJ IDEA

# 3. Execute as classes de uma sessão (exemplo: camada sessão 1)
javac (Get-ChildItem "camada sessão 1\src\*.java").FullName
java -cp "camada sessão 1\src" NomeDaClasse
```
